## 參數設定

![參數設定](https://raw.githubusercontent.com/lya79/CocosGobangoV2/master/1.PNG)

參數名稱 | 說明
--- | ---
Level | 0:設定電腦隨機下棋, 1:電腦會基礎攻擊和防禦
Bingo | 設定幾顆棋子連線獲勝
Row | 棋盤列數
Col | 棋盤行數
Rect Opacity | 棋盤格子透明度
Rect Offset | 棋盤向下偏移量
Rect Width | 棋盤格子寬度
Rect Height | 棋盤格子高度
Player Width | 棋盤角色寬度
Player Height | 棋盤角色高度
Rect Spacing | 棋盤格子間距
